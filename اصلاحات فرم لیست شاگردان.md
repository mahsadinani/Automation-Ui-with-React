# اصلاحات فرم لیست شاگردان

## لیست تغییرات مورد نیاز:

1. **اضافه کردن چک باکس نوع کلاس**: یک چک باکس با مقادیر (خصوصی / گروهی) اضافه کن

2. **اضافه کردن چک باکس رزرو**: یک چک باکس به عنوان رزرو اضافه کن. دیفالت باید این تیک بخورد. اگر تیک را برداشت، فیلد "کد کلاس" اضافه شود.

3. **توسعه عملیات**: در قسمت عملیات امکان (ویرایش / حذف / اضافه کردن به کلاس / صدور گواهینامه / ساخت رکورد مشابه / پروفایل مالی) اضافه شود.

4. **حذف فیلد وضعیت**: فیلد وضعیت حذف شود.

5. **تنظیم فیلدهای الزامی**: تنها فیلد نام و نام خانوادگی الزامی باشد.

6. **ادغام فیلدها**: فیلد "دوره" و "کلاس" باید یکی شوند.

7. **حذف فرم ساده**: فرم ساده رو بردار و فقط فرم پیشرفته رو نگه دار.

8. **حذف فیلد مبلغ پرداخت**: فیلد مبلغ پرداخت رو حذف کن.
